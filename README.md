# Mathematical Pedagogy: Convergent Educational Platform

This platform is a practical implementation of a doctoral research project (Specialty 13.00.04 - Theory and Methodology of Professional Education). It is designed to transform the professional training of future mathematics teachers by integrating fundamental mathematics with modern digital tools.

## 🚀 Key Features
- **Tensor-Based Competency Model:** A multi-dimensional (3D) approach to mapping mathematical, technological (Python/R/Desmos), and pedagogical skills.
- **Adaptive Learning Trajectories:** Automated generation of individual paths using **Tensor Factorization (CP-decomposition)** to analyze student performance.
- **Vibe-Coding Paradigm:** Leveraging Generative AI to reduce syntax barriers, shifting focus from coding mechanics to mathematical semantics.
- **Convergent Content:** A series of 8 modules covering everything from Ancient Greek geometry to Differential Equations and Data Science.

## 🛠 Tech Stack
- **Core:** Python (Streamlit, NumPy, TensorLy)
- **Visualization:** Desmos API, Matplotlib
- **Analytics:** R (tidyverse) for statistical verification
- **Deployment:** GitHub & Streamlit Cloud

## 📚 Scientific Context
The platform serves as a "Convergent Intellectual Environment" that fosters the development of computational thinking and professional adaptability in pre-service mathematics teachers.
